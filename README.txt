	ZHarvester - проект, который реализует протокол Z39.50 для доступа харвестера бибилиотек.
	
	Для запуска можно собрать проект из папки ZHarvester используя файл сборки META-INF/build.xml
	
	Для удобства прикреплен собранный вариант в папке compiled. ZHarvester.jar требует наличия паки libs
	ZHarvester.jar - выполняемый файл. Будет обращение к серверу opac.ruslan.ru:210/lounb с сохранением 
		файлов в папку results
		
	Для тестирования можно открыть проект из папки Test в методе main единственного класса которого 
		вызывается метод executeClassMethod который иденитичен одноименному методу из харвестера. 
		Список полей класса Query указаны в комментариях перед методом main.
	
	Небольшой список доступных серверов лежит в корне репозитория. Справа от URL адреса указано 
		количество бибилиографических записей, которое вернул сервер по запросу с языком rus
